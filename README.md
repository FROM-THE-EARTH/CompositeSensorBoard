# CompositeSensorBoardLib
FTE製の複合センサ基板です．開発は主にEAGLEを利用して行っています．

## 複合センサのメリット

- 市販のセンサを個別に買うより低コスト
- 無駄な回路を省けるので小型化が可能
- インターフェースを揃えているので使い回しが可能
